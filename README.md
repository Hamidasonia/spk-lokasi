## SPK Pemilihan Lokasi menggunakan metode MFEP

# Install Dependency
```console
$ composer install
$ npm install
```

# Database Migration
```console
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
```

# Start Server
```console
$ php artisan serve
```
